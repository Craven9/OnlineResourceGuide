Document Structure Tags

<!DOCTYPE html>

Declares the document type and version of HTML

<!DOCTYPE html>

<html>

Root element of an HTML document

<html lang="en">
</html>

<head>

Contains metadata about the document

<head>
    <title>Page Title</title>
</head>

<body>

Contains the visible content of the document

<body>
    <h1>Visible content</h1>
</body>

Text Formatting Tags

<h1> to <h6>

Heading elements (h1 is largest, h6 is smallest)

<h1>Main Heading</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>
Output:

Main Heading

Section Heading

Subsection

<p>

Paragraph of text

<p>This is a paragraph.</p>
Output:

This is a paragraph.

<strong>

Important text (usually bold)

<strong>Important text</strong>
Output:
Important text

<em>

Emphasized text (usually italic)

<em>Emphasized text</em>
Output:
Emphasized text

<br>

Line break (self-closing)

First line<br>
Second line
Output:
First line
Second line

<hr>

Horizontal rule/line (self-closing)

Content above
<hr>
Content below
Output:
Content above
Content below

List Tags

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
Output:
  • Item 1
  • Item 2

<ol>

Ordered (numbered) list

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>
Output:
  1. First item
  2. Second item

Container Tags

<div>

Generic block container

<div class="container">
    Content here
</div>
Output:
Content here

<span>

Generic inline container

<p>This is normal text with 
<span class="highlight">highlighted text</span> inside.</p>
Output:

This is normal text with highlighted text inside.

<header>

Header section of a page or section

<header>
    <h1>My Website</h1>
    <nav>Home | About | Contact</nav>
</header>
Output:

My Website

Home About Contact

<main>

Main content area

<main>
    <article>Main content</article>
</main>
Output:
Main content

<footer>

Footer section

<footer>
    <p>Copyright information</p>
</footer>
Output:

Copyright information

<section>

Thematic section of content

<section>
    <h2>Section Title</h2>
</section>

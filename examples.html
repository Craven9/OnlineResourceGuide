Complete HTML Page Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Blog Post</title>
    <meta name="description" content="Learn about HTML basics">
</head>
<body>
    <header>
        <h1>My Blog</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>HTML Basics</h2>
            <p>Published on <time datetime="2025-11-11">November 11, 2025</time></p>
            <p>This is a complete example of an HTML page.</p>
        </article>
    </main>
    
    <footer>
        <p>&copy; 2025 My Blog</p>
    </footer>
</body>
</html>
Output:
My Blog Post

My Blog

  • Home
  • About

Table Example

Article with Semantic HTML

<article>
    <header>
        <h1>Understanding HTML5 Semantic Elements</h1>
        <p>By <strong>John Developer</strong> on <time datetime="2025-11-11">November 11, 2025</time></p>
    </header>
    
    <section>
        <h2>What are Semantic Elements?</h2>
        <p>Semantic elements clearly describe their meaning to both the browser and the developer.</p>
        
        <blockquote cite="https://www.w3.org/TR/html52/">
            "Semantic elements give meaning to the content they contain."
        </blockquote>
    </section>
    
    <section>
        <h2>Benefits</h2>
        <ul>
            <li>Better accessibility</li>
            <li>Improved SEO</li>
            <li>Cleaner code</li>
        </ul>
    </section>
    
    <footer>
        <p><small>This article is part of our HTML tutorial series.</small></p>
    </footer>
</article>
Output:

Understanding HTML5 Semantic Elements

By John Developer on November 11, 2025

What are Semantic Elements?

Semantic elements clearly describe their meaning to both the browser and the developer.

"Semantic elements give meaning to the content they contain."

Benefits

  • Better accessibility
  • Improved SEO
  • Cleaner code

HTML Attributes Examples

<!-- Links with different targets -->
<a href="https://example.com" target="_blank" rel="noopener">External Link</a>
<a href="#section1">Internal Anchor</a>
<a href="mailto:contact@example.com" title="Send us an email">Contact Us</a>

<!-- Images with different attributes -->
<img src="logo.png" alt="Company Logo" width="200" height="100">
<img src="responsive.jpg" alt="Responsive design" loading="lazy">

<!-- Form inputs with validation -->
<input type="text" name="username" required minlength="3" maxlength="20">
<input type="email" name="email" placeholder="Enter your email">
<input type="password" name="password" pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}">

# HTML Cheat Sheet and Resource Guide - Site Map

## WordPress Setup Recommendations

**Theme Qualities:**
- Clean, readable typography with good contrast
- Responsive design for mobile/tablet viewing
- Good heading hierarchy styling (H1, H2, H3)
- Code block formatting support
- Skip-to-content and accessibility features
- Fast loading times

**Menu Structure:**
```
Primary Menu:
- Home
- HTML Quick Reference
- Six Essential Resources  
- Study Tools
- Help & FAQ
```

**Page Structure and Content:**

## Page 1: Home (Landing Page)
**File:** index.html
**Menu Label:** Home
**Content:**
- Hero section with guide title and purpose
- Table of Contents (2-level) with anchor links
- What's in the Guide
- Who the Guide is for
- What the Guide assumes users know
- How the Guide is organized for use
- Tips for using the Guide

## Page 2: HTML Quick Reference
**File:** html-reference.html  
**Menu Label:** HTML Quick Reference
**Content:**
- Basic page structure
- Text and structure tags
- Links and images
- Semantic layout elements
- Tables and forms basics
- Common attributes
- Common beginner mistakes
- Mini templates
- Quick Reference Index (alphabetical)

## Page 3: Six Essential Resources
**File:** resources.html
**Menu Label:** Six Essential Resources
**Content:**
- Introduction to resource selection criteria
- Six fully developed resource abstracts with:
  - Title, URL, Category
  - Complete abstracts (what, content, relevance, usage tips)
  - Concrete examples
  - Quality evaluation

## Page 4: Study Tools
**File:** study-tools.html
**Menu Label:** Study Tools  
**Content:**
- Printable cheat sheet
- Glossary of terms
- Assignment submission checklist
- Practice exercises

## Page 5: Help & FAQ
**File:** help-faq.html
**Menu Label:** Help & FAQ
**Content:**
- FAQ for common beginner confusion
- Debugging tips
- Troubleshooting common errors
- Contact information

**WordPress Categories/Tags (if using posts):**
- Categories: HTML Basics, Resources, Tools, Troubleshooting
- Tags: beginner, web development, reference, tutorial, validation

**Internal Linking Strategy:**
- Cross-reference between sections
- Anchor links within long pages
- "Related" sections at page ends
- Consistent navigation footer on all pages
